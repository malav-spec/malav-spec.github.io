Cache Simulator

This is a 2 player game based on the concept of Chess. In this game one of the players is an AI agent and the other is a human player. The AI agent uses Markov Decision Process to find out the next best step. The project is done in Java.

Designing a PThread Library and Thread Scheduler

The project implements the pthread library in C and a scheduler for scheduling all the threads. The scheduler uses the Preempting Shortest Job First (PSJF) algorithm to order the threads. The library has all the functions of the POSIX thread library.
